Project Sosial Media sederhana yang dikembangkan dengan tech :
  1. MongoDB => Database
  2. Express js => Backend
  3. React js => Frontend
  4. Node js
  5. Joi => validation
  6. JWT => auth (Authentication & Authorization)
  7. Nodemailer => mengirim kode otp
  8. bcript => hash password user

Service dari pihak ketiga sebagai penunjang aplikasi:
  1. Google Drive => penyimpanan file gambar
  2. SendGrid => layanan untuk mengirim email (kode OTP)
